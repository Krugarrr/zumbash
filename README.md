# Zumbash

Недоразвитый клон игры Zuma, написанный на чистом bash

## Quickstart 

Чтобы скачать и сразу запустить игру, используйте в терминале Bash следующую команду:

```bash
curl -O https://raw.githubusercontent.com/Krugarrr/zumbash/main/zumbash.sh && chmod +x zumbash.sh && ./zumbash.sh
```

## Суть игры
1. Если сложить в ряд 3 или более шаров одинакового цвета - они пропадут, и вам начислятся очки
2. Вы побеждаете в том случае, если уничтожите змейку из шаров до того, как она доберётся в лунку
3. Если змейка доберётся в лунку, вы проиграли

## Управление
A / D - Движение пушки влево и вправо

W / ПРОБЕЛ - Выстрел. Можно стрелять сразу несколько раз

Q - Выход
